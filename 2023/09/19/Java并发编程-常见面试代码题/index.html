Java并发编程-常见面试代码题

题目1

  • 题目描述
  • Code 如果是三个线程循环打印0-100呢？修改下while的条件即可，实际上还是唤醒了所有阻塞进程，不过每个阻塞进程会去判断当前其是否可以继续运行。 循环打印abc，写法一样，修改下判断条件即可。 这样的写法是唤醒所有的线程，然后线程去根据当前共享变量的状态决定是否阻塞，能否修改唤醒指定的线程？
    需要为每个线程提供单独的锁对象。
    这种方式会出现死锁，目前还不知道原因。
